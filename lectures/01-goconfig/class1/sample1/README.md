## 展示内容

- 使用 goconfig 包加载及保存 INI 格式的配置文件
- 展现 goconfig 对同时支持注释读写操作的强大特性
- 通过 goconfig 包对指定键进行读写操作，包括进行类型自动转换和 Must 系列方法
- 通过 goconfig 对键值进行删除的操作
